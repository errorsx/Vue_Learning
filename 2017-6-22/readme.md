# 模版语法 Mustache 语法

*  文本

v-once  <br />
执行一次性地插值，当数据改变时，插值处的内容不会更新,会影响到该节点上所有的数据绑定<br />

*  纯HTML

v-html  <br />
双大括号会将数据解释为纯文本，而非 HTML 。为了输出真正的 HTML ，你需要使用 v-html 指令<br />


###  你的站点上动态渲染的任意 HTML 可能会非常危险，因为它很容易导致 XSS 攻击。请只对可信内容使用 HTML 插值，绝不要对用户提供的内容插值。
